Testing Instructions

  1. Open the browser console (F12 or Cmd+Option+J)
  2. Look for messages about autumn leaves initialization
  3. If it's fall (Sept-Nov), you should see leaves falling
  4. If it's not fall, you'll see a message that the animation is disabled

To Test Other Seasons

Run this in the console to temporarily override the date:

Date.prototype.getMonth = function() { return 9; }; location.reload();

(9 = October, 8 = September, 10 = November)
